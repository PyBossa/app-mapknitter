MapKnitter tutorial

Hi! This is a demo application that shows how you can analyze MapKnitter maps using the PyBossa framework in Crowdcrafting.org.

The application is really simple. It loads a map from the MapKnitter repository, centers the map and the goal is to browse the map and mark with points all the visible tents.

The web map provides different layers so you can change between them in order to get some help from satellite images.

When you find a tent, click in the Point button and place it in the middle of the park. Then click submit and you will have completed one task.

As this is a demo application, you can also draw polygons. Polygons could be used to measure areas in the maps.
